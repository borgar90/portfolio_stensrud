# Borgar's Portfolio - Full-Stack Developer

A modern, responsive portfolio website showcasing 18+ years of full-stack development experience. Built with Next.js, TypeScript, and Tailwind CSS using VS Code's dark stone theme with purple accents.

## 🚀 Features

- **Hero Section**: Engaging introduction with animated background and call-to-action buttons
- **Technical Skills**: Interactive showcase of programming languages, frameworks, and tools
- **Experience Timeline**: Professional journey with detailed accomplishments and technologies
- **Project Portfolio**: Showcase of featured projects with detailed descriptions and tech stacks
- **Additional Skills**: Soft skills, methodologies, and professional competencies
- **Contact Form**: Professional contact section with form and social links
- **Responsive Design**: Mobile-first design that works perfectly on all devices
- **Smooth Animations**: Framer Motion animations for engaging user experience
- **VS Code Theme**: Dark stone color palette with purple accents

## 🎨 Design Features

- **VS Code Dark Stone Theme**: Authentic development environment colors
- **Purple Accent Colors**: Beautiful gradient purple highlights
- **Near-White Text**: High contrast for important elements
- **Smooth Animations**: Entrance animations and hover effects
- **Interactive Elements**: Hover states and click animations
- **Professional Typography**: Inter and Fira Code fonts

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## 🔧 Customization

To customize the portfolio for your own use:

1. **Update Personal Information**: Edit hero section, experience, and contact details
2. **Modify Skills**: Update the skills arrays with your technologies
3. **Add Projects**: Replace projects with your own work
4. **Change Colors**: Modify the Tailwind config for different color schemes
5. **Update Content**: Customize all text content to match your experience
